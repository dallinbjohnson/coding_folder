VersaLink B610 – Overview

Fast, full featured and office friendly, the VersaLink B610 Printer is everything you need in a black and white printer. It features Xerox ConnectKey Technology, which sets a new standard in workgroup productivity. It is highly configurable, mobile-ready, cloud connected, app-enabled and customisable. VersaLink B610 – the affordable, modern workplace assistant.

Easy, Smart and Secure

An entirely new design, with fewer moving parts, strengthened paper-path components and an advanced Hi-Q LED imaging system it will ensure reliability day in and day out.

With IT-free installation wizards to step-by-step configuration options, this new device can be set up quickly and easily. Once set up, you can count on its superior print quality, device efficiency and benchmark security.

Customisable, App-based Functionality

From the intuitive, 5-inch colour touch screen UI to the downloadable Xerox ConnectKey Apps from the App Gallery, the VersaLink B610 series is fully customisable and personalisable, to optimise office efficiency.

Cloud and mobile connected

Easy mobile connection includes Apple® AirPrint®, Google Cloud Print™, and Mopria®. It has Near Field Communication (NFC) Tap-to-Pair as standard to easily link your mobile device to the printer, plus optional Wi-Fi and Wi-Fi Direct.

Add the finisher with 500-sheet output tray and 50-sheet stapling.

Add the optional Wi-Fi and Wi-Fi direct Kit
